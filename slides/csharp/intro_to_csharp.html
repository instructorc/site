Context

c#
is a simple, modern, object-oriented, and type-safe programming language

Developed as an object-oriented and component-oriented language

Modeled after the C++ programming language

.NET Core
is a cross-platform .NET implementation for websites, servers, and console apps on Windows, Linux, and macOS.

Dot.net core Guide

Includes easy-to-use command-line tools that can be used for local development and in continuous-integration scenarios.

.NET Framework
A programming infrastructure created by Microsoft for building, deploying, and running applications and services that use .NET technologies, such as desktop applications and Web services.

The .NET Framework contains three major parts

  • The Common Language Runtime
    • converts compiled code into machine instructions which the computer's CPU then executes
  • The base class library
    • a collection of reusable classes, interfaces and value types
    • .NET Core API Reference
    • .NET Framework API Reference
  • Other managed libraries suchs as ASP.NET

Program Components

  • Namespace
    • Provides a way to group similar classes
    • Can be used to avoid naming conflicts in large projects
  • C# method parts
    • Method header
      • Includes the method name and information about what will pass into and be returned from a method
    • Method body
      • Contained within a pair of curly braces and includes all the instructions executed by the method
  • Access Modifier
    • Defines the circumstances under which the method can be accessed
    • Public, private, protected, internal, readonly, etc..
  • Keywords
    • Predefined and reserved identifiers that have special meaning to the compiler
  • The name of the method is Main()
    • Every application must have a Main() method
    • Classes with a Main()method are called application classes; others are non-application classes
  • The method returns nothing as indicated by the keyword void

Compiling and Executing a C# Program

  • Steps for viewing a program's output
    • Compile source code into intermediate language(IL)
    • The C# just in time(JIT) compiler translates the intermediate code into executable statements
  • You can either of two ways to compile
    • The command line
    • The Integrated Development Environment (IDE)

Sample Program to Compile

  • Use any text editor to write the following code and save as tuition.cs
  • Why string[] args?
							
								using System;
								using System.Text;
								
								class Program{
									static void Main(string[] args)
									{
									
									/**first name and credit hours should be provided 
									as command line arguments**/
										
									  const double TUITION = 985.00;
									  double tuitionCost;
										
									  tuitionCost = TUITION * Convert.ToDouble(args[1]);
										
									  Console.WriteLine(args[0] + " your tuition cost is " + tuitionCost.ToString("c"));
										
									}
								}

Steps to compile via command line

Step 1
Open the Developer Command Prompt VS 2017

Step 2
Change to the directory where program is located

Step 3
Type the csc tuition.cs command to compile

Step 4
type the Dir command to view files (this step is not necessary)

Step 5
Type tuition and press Enter to run the program
