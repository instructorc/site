What is A/B Testing and Controlled Experiments?

A/B testing—also called split testing or bucket testing—compares the performance of two versions of content to see which one appeals more to visitors/viewers. It tests a control (A) version against a variant (B) version to measure which one is most successful based on your key metrics.

Benefits of A/B testing (Controlled Experiments)

Optimization
Can measure if feature is having impact; impact performance

Alignment
Can help align business objectives with KPI's; impact ROI

User Experience
Can refine user experience; Insight into user preference and behavior

Continous Improvement
Can help foster a culture of improving product; Also helps mitigate risk

Data-Driven
Can allow for informed decision making; minimizes assumption

Innovation
Can produce a culture of entreprenural thinking; testing new ideas

A/B Testing Process

1. Problem Statement
Understand the business Goal - Define the User Journey - Identify the product experience you want to achieve - Specify Success/Key Metric's

2. Hypothesis Testing
State the Hypothesis:
Null Hypothesis - the hypothesis that there is no significant difference between specified populations.
Alternative Hypothesis - There’s an effect in the population.
Set significance levelSet statistical power

3. Design the Experiment
Set the randomization unit - Identify target Populaton - Determine the Sample Size - Set Duration of the Experiment

4. Run the Experiment
Set up Instruments and data pipelines to collect data - Avoid peeking p-values

A/B Testing Process Cont.

5. Validity Checking
Gaudrail Metrics - External Factors - Selection Bias - Novelty Effect

6. Intrepret the Results
Statistical Significance - p-Value is greater thatn .05

7. Launch Decision
Determine the cost of launching

Success Metric

Measurable
Can the behavior be tracked using the data you collect from users?

Attributable
Can you assign the behavior(effect) to the treatment(cause)

Sensitive
Does the metric have low variability that you can distinguish the treatment from the control?

Timely
Can you measure the success behavior in a short term?
