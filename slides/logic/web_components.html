What are Web components ?

A collection of web standards that makes it easy to create reusable objects with regular HTML and JavaScript

  • We already use them in Standard HTML but natively
  • A lot of popular GUI tools are build in a component based structure

HTML Imports

Allows HTML markup content to be defined as an external file and then included into other webpages. Things are still moving with this standard

  • Sitepoint write-up
										
											
		
		
										
									
										
											var htmlImport = document.querySelector('link[rel="import"]');
											var htmlDoc = htmlImport.import;
											var htmlMessage = htmlDoc.querySelector('.message-success');
											document.body.appendChild(htmlMessage.cloneNode(true));
