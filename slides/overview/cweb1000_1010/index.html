First Day Presentation

Covers the class syllabus, course expectations, and communication.

Web Tools Presentation

Covers development tools and environments (text editors, IDEs, and local vs cloud). Also covers file structures. and goes over absolute vs relative file paths.

HTML5 Presentation

Discusses the history of the Web and HTML, the World Wide Web Consortium and its purpose, and covers the rules and syntax of HTML. Also covers the categories of HTML5 elements, the structuring of a basic HTML element, and validating documents.

Introduction to CSS Presentation

Covers CSS style rules, the different types of style sheets, selectors and attribute selectors, inheritance in CSS, and implementing pseudo-classes and pseudo-elements.

Data Modeling

Articulate the purpose of a database Explain the difference between data, information and knowledge Install database development software and tools Analyzing various Data Models Conceptual | Logical | Physical

Introduction to Site Planning and UX

Discusses the differences between UI and UX, why user research is important. information architecture and navigation, usability testing, accessibility, and wireframes and prototyping. Also covers low-fidelity vs high-fidelity wireframes, and the tools used to create them.

Web Page Layout Presentation

Covers the differences between fixed, fluid, adaptive, and responsive layouts. Also covers the components of box modeling such as padding, margin, borders, etc. Covers a traditional float layout, CSS flexbox, and CSS grid.

Color, Graphics, and Typography

Covers the basics of understanding the use of color and best color practices. Discusses the different file formats for graphics, reviews CSS background properties, and covers typography and fonts.

Implementing Responsive Design

Discusses why responsive designs are used, and the differences between responsive and adaptive designs. Covers CSS media queries.

Animation and Transitions

Covers CSS transitions, animations and keyframes, and animation libraries.

Introduction to JavaScript

Covers the basics of JavaScript and client/server architecture. Reviews how to incorporate JavaScript, how to write a program, JavaScript objects, and how to debug and troubleshoot your code.

JavaScript Functions Presentation

Covers how to implement a JavaScript functions and objects, and covers an example of APIs using the Google Maps API.
